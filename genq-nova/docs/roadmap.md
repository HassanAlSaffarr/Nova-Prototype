# Nova — What's Next

The prototype proves the loop: satellite change detection over Karrada, fused
with multi-agent signals on one map. Here's how Nova grows from demo to product.

## 1. Sharper detection
Add **Sentinel-1 radar**, which sees through cloud and haze, so we never miss a
change waiting for a clear sky. For the highest-value sites, confirm with
**sub-metre imagery** (Planet, Maxar). Validate our vegetation/built-up
thresholds against known ground truth to put a real accuracy number on every
detection.

## 2. Always-on pipeline
Run Nova automatically every few days as fresh Sentinel-2 imagery lands. Each run
**updates** existing sites instead of duplicating them, so we build a timeline of
how each location changes — not just a one-off snapshot.

## 3. More cities
Extend beyond Karrada to **Basra, Erbil, Mosul, and Najaf**, with detection
thresholds tuned per city. Once several cities are live, Nova can spot
**patterns across them** — where development is accelerating, and where it stalls.

## 4. Verification
Automatically cross-reference each detection against **building permits, news,
and social chatter**, turning a satellite signal into a corroborated event. An
**analyst review queue** lets a human confirm or dismiss the highest-value finds.

## 5. Building-level intelligence
On high-resolution imagery, classify **building types** and attach attributes —
height, use, approximate age — to each footprint, so the map answers not just
"something changed" but "what is it."

## 6. Real agents
Replace the demo's synthetic agents with live data: **Roberto** from field
surveys, **Namroud** from institutional and financial filings, **Peter** from
social platforms — each feeding the same unified map.
